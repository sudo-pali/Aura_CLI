# Aura Farming

Saiba se sua aura é W:

- Pressione "Enter" após inicializar o programa com o comando de uso abaixo

-  Duvido acharem o "Easter Egg" da Aura negativa (Dica: tem haver com o botão "Enter", ou a ausência dele hehe)
